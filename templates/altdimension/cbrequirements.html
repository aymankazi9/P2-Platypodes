Big Idea Alignment
Creative Development This project has been built up from the collaboration of 4 team members. All work was split between frontend development and backend development to make working as efficient as possible. Errors often show up due to css overlapping therefore, correction of these aesthetics require collaboration and insight into program function/design.
Data All data found on website tracking COVID-19 was scraped from other websites and put into neat lists for our website to use.
Algorithms and Programming Various algorithms used in procedures to extract information from data as well as orienting information to provide aesthetic data. Main uses include lists, iterations, variables, assignments, and libraries to organize all these algorithms and use when required.
Computer Systems and Networks Knowledgeable on computing, however, our server runs on a singular computer. Parallel processing powers are not required for this website. In case it is necessary, we are aware of how to set that up.
Impact of Computing We understand where data comes from and how to analyze its trends. The hotspot pages show sense of understand on how to interpret data as well as how to analyze trends in that data.
